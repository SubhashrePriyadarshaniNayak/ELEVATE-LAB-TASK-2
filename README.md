# Task 2: Data Visualization and Storytelling

## 📊 Project Overview

This project focuses on creating a data visualization dashboard using Power BI. The dataset used is the Sample Superstore dataset, which contains sales, profit, and regional data.

## 📁 Files Included

* Sample Superstore.xlsx (Dataset)
* Task2Submission.pbix / .pbi (Power BI file)
* Dashboard Screenshot (visual output)

## 📈 Dashboard Features

* KPI Cards: Total Sales, Total Profit, Total Orders
* Sales and Profit analysis by Region
* Profit analysis by Category
* Sub-category analysis to identify loss-making products
* Sales trend over time using line chart

## 🔍 Key Insights

* West region has the highest sales and profit
* Technology category performs best in terms of profit
* Some sub-categories are generating losses
* Sales vary over time, showing trends and patterns

## 🛠 Tools Used

* Power BI

## 📌 Conclusion

The dashboard provides clear insights into business performance. It helps identify profitable areas, loss-making products, and overall sales trends, which can support better decision-making.
